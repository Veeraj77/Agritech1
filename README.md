
# 🌾 AgriTech AI – Smart Farm-to-Market Assistant

AgriTech AI is an intelligent agriculture platform that helps farmers optimize crop yield, monitor health, and improve farm-to-market logistics.  
This system uses AI models to provide **yield prediction**, **market price forecasting**, **storage suggestions**, and **logistics route optimization**.

---

## 🚀 Features

### 🔍 1. Crop Yield Prediction  
Enter soil data, rainfall, fertilizer levels, and the system predicts crop yield using ML models.

### 🌱 2. Crop Health Monitoring  
AI-based detection using environmental and crop input data.

### 📈 3. Market Price Forecast  
Predicts future market prices using trained ML models deployed on HuggingFace.

### 🛣️ 4. Farm-to-Market Logistics  
Optimizes best route, distance, and transport suggestions.

### 📦 5. Storage Recommendation  
Suggests ideal storage methods based on crop type and climate.

### 🎨 6. Modern UI  
Built using **React + Vite + Tailwind + ShadCN UI** with smooth animations and an elegant green Agri theme.

---

## 🛠️ Tech Stack

### **Frontend**
- React 18  
- Vite  
- TailwindCSS  
- ShadCN UI  
- Lucide Icons  
- Framer Motion  
- React Router  

### **Backend / AI Models**
- Python ML Model (Random Forest)  
- Rule-Based Model  
- HuggingFace Deployment (`/predict` endpoint)

### **Hosting**
- Frontend → **Vercel**  
- Backend Model → **HuggingFace Spaces**
